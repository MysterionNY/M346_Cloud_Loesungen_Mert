# KN03 IaaS - Virtuelle Server

## Aufgabe 1 - Installation von Web- und Datenbankserver
Ich habe die Commands in der Abfrage des KN03 ausgeführt. Nachdem ich meinen User mit meinem selbstgesetzten Passwort erstellt habe, musste ich diese entsprechend anpassen und zuweisen.

```bash
sudo nano /var/www/html/db.php
```
Dort drinne habe ich mein passwort hineingeschrieben und abgespeichert. Dies hat mir die Möglichkeit gegeben, auf die DB erfolgreich zuzugreifen.

![Index Page][index]
![Info Page][info]
![DB Page][db]
![Instanz][instanz]
![Instanz Regeln][regeln]

```bash
mysql -u admin -p
```
Mit Eingabe von diesem Command kann ich mich mit meiner DB verbinden. Danach muss ich das Passwort angeben und habe Zugriff. 

Mit dem Select statement
```bash
SELECT Host, User FROM mysql.user;
```

werden alle Benutzerkonten ausgegeben, die in der MariaDB vorhanden sind. Die gelisteten Informationen zeigen an welche User es gibt und welche hosts sich verbinden dürfen
